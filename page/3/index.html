素数和因子

发表于 2020-09-08 更新于 2020-10-23 分类于 数学 Valine：

素数是构成正整数大厦的基石

  • 真因子:每个自然数$a$至少有$1$和它本身$a$。如果$b|a$且$b\neq1或a$，那么我们称$b$是$a$的一个真因子.
  • 素数:一个大于1的自然数没有真因子，那么它就是素数。

判断一个数的因数只需要除到它的一半

确定一个数是不是素数(素性检验)是一项非常艰巨的任务，特别是当数字比较大的时候。但是还是有些小技巧可以用，稍微想一下我们可以知道，其中有一半的数是不必要检验的:如2，3，···，96｜97，这些数中大于$\frac{97}{2}$的数不整除97。证明这一点很简单，假定$b\gt \frac{97}{2}$且$b$是97的一个真因子，我们将看见这一假定导出一个矛盾。设$97=bc$。这里$c$是97的一个真因子，因而$c\geqslant 2$，于是$97=bc\geqslant b \times 2 \gt \frac{97}{2}\times 2=97$我们得到$97\gt97$，矛盾。所以大于$\frac{97}{2}$的数不会整除97。

CSS学习之清除浮动

发表于 2020-09-04 更新于 2020-10-23 分类于 计算机 Valine：

为什么需要清除浮动？

  • 父级没高度;
  • 子盒子浮动了;
  • 影响下面布局了。

清除浮动本质

  • 清除浮动造成的影响
  • 如果父盒子本身有高度，则不需要清除浮动
  • 清除浮动之后，父级就会根据浮动的子盒子自动检测高度，父级有了高度，就不会影响下面的标准流了。

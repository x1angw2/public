Shell脚本之结构化命令

本文主要记录Shell脚本之结构化命令学习记录。

if-then

最基本的结构化命令:

当if语句中的命令返回退出状态码为0时，then部分中的命令会被执行。非0时，则不执行。fi为该命令结束标志。

if-then-else

当if语句中的命令返回退出状态码为0时，then部分中的命令会被执行。非0时，则执行else部分中的语句。

嵌套if

太麻烦，不常用。常用的为下：

如果elif返回的状态码为0，则执行第二个then语句。

记住，在elif语句中，紧跟其后的else语句属于elif代码块。它们并不属于之前的if-then代码块。

可以继续将多个elif语句串起来，形成一个大的if-then-elif嵌套组合

tset命令

if只能根据命令退出码来决定是否执行then语句。可以利用test命令测试其它条件。test condition

如果test后不写任何判断条件，它的返回值为非零。
更为常见的写法是无需在if-then语句中声明test命令

注意[ ]内容的前后必须分别加一个空格。

test命令可以判断三类条件：

  • 数值比较
  • 字符串比较
  • 文件比较

数值比较功能

比较 描述
n1 -eq n2 检查n1是否与n2相等
n1 -ge n2 检查n1是否大于或等于n2
n1 -gt n2 检查n1是否大于n2
n1 -le n2 检查n1是否小于或等于n2
n1 -lt n2 检查n1是否小于n2
n1 -ne n2 检查n1是否不等于n2

使用变量赋值时只能比较整数，浮点数会报错。

字符串比较功能

比较 描述
str1 = str2 检查str1是否和str2相同
str1 != str2 检查str1是否和str2不同
str1 < str2 检查str1是否小于str2
str1 > str2 检查str1是否大于str2
-n str1 检查str1的长度是否非0
-z str1 检查str1的长度是否为0

需要注意的是使用>和<符号时，要用到转义符/。否则shell会把它们当作重定向符号。

文件比较功能

比较 描述
-d file 检查file文件是否存在并是一个目录
-e file 检查file文件是否存在
-f file 检查file文件是否存在并是一个文件
-r file 检查file文件是否存在并可读
-s file 检查file文件是否存在并非空
-w file 检查file文件是否存在并可写
-x file 检查file文件是否存在并可执行
-O file 检查file文件是否存在并属当前用户所有
-G file 检查file文件是否存在并且默认组与当前用户相同
file1 -nt file2 检查file1是否比file2新
file1 -ot file2 检查file1是否比file2旧
